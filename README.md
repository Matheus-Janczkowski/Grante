# Grante
Repository to store the Grante's Wonderland.

Booklet link: https://www.overleaf.com/read/wbxhncmtnmkm#0eb73b

# Installation
Download the repository, unzip the file, put it in a suitable place for you.
Activate a python virtual environment (follow instruction in the booklet to
create a virtual environment if you don't have one), go into the directory 
where the files are located through the virtual environment terminal. Then, 
type in terminal:

python3 setup.py bdist_wheel sdist

pip install .

To test the installation:

python3

import Grante
